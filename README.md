# Controls
### In Game Controls
| Key       | Description           |
| --------- | --------------------- |
| P         | Pause Obstacles       |
| SHIFT + P | Pause Background      |
| O         | Spawn Obstacles       |
| SHIFT + O | Spawn Background Wave |
| 0         | Toggle Free Cam       |
| W         | Jump                  |
| A         | Move Left             |
| D         | Move Right            |

### Free Cam Controls
| Key           | Description  |
| ------------- | ------------ |
| W             | Look Up      |
| S             | Look Down    |
| A             | Look Left    |
| D             | Look Right   |
| Space         | Move Forward |
| Space + SHIFT | Move Forward |
